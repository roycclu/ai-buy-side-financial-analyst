Architecture
Five-Agent Research Pipeline

Five specialized agents run sequentially, each handing structured artifacts to the next stage. No human input required between phases.

Agent 0 — Project Manager

Setup & Routing

Collects the research question and company list from the user, validates inputs, and initialises the pipeline. Writes project config to projects.json.

Agent 1 — Research Specialist

SEC EDGAR Filing Collector

Fetches 10-K and 10-Q filings from SEC EDGAR. Cache-first: skips download if filings already exist locally. Handles CIK lookup, search, and rate limiting.

Agent 2 — Financial Analyst

Per-Company Extractor

One isolated LLM session per company. Reads the two most-recent filings and outputs CompanyFacts JSON, CompanyBrief markdown, and QuoteBank JSON — parsed from structured XML output, not LLM tool calls.

Agent 3 — Lead Analyst

Investment Report Writer

Reads compact summaries from Agent 2. Writes per-company analyst reports with thesis, margin analysis, AI/CapEx view, risks, and stance. Produces a sector synthesis answering the research question.

Agent 4 — Visualization

Chart Generation

Consumes viz specs from Agent 3, reads structured KPI JSON files, and generates bar, line, and comparison charts as PNG files in the visualizations folder.
